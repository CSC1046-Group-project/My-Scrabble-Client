# My-Scrabble-Client
Java Client for MyScrabble game

## How to run ?
The following documentation will help you to run the client using Maven. Be sure you have the most recent java version.

### Run using Maven
If you have Maven installed on your computer, you can compile the project at the root of the repository using:
```
mvn compile
```

Then run the project using:
```
mvn exec:java
```
